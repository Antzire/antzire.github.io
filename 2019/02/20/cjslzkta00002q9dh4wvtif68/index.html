Linux Install Oracle11g GoldenGate

发表于 2019-02-20 | 分类于 GoldenGate

GoldenGate公司是专注于数据同步领域，是实现数据同步技术的领导者，在2009年被Oracle公司收购，GoldenGate公司的领先技术就是逐渐为大家所知的交易数据管理Transactional Data Management (TDM)，可以在异构的IT基础结构之间实现大量数据的秒一级的数据捕捉、转换和投递。GoldenGate可以支持几乎所有常用操作系统和数据库平台。下面我们进行Oracle GoldenGate简称OGG的安装，它的安装非常简单，前提是已经安装了对应版本的数据库，目前我使用的OGG版本支持Oracle 11g、12c两个版本，OGG的数据复制模式、配置过程在后续的文章中更新。

1. 资源描述

  • GoldenGate 组件版本选择为12.3.0.1
